## 2. 命令
- 创建环境并安装openai软件包
```sh
conda create -n deepseek python=3.11 openai
```
- 激活环境
```sh
conda activate deepseek
```
- 环境变量修改
```sh
export PATH = /path/to/python:$PATH
```
